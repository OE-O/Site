Features
The OE-O Launcher has many features similar to the steam workshop.
Auto Updates

The launcher checks every half hour for updates on each mod using a simple version to version check system.

Simple UI

The launcher has a simple UI, easy enough for you to look around and install mods for your games.

Wide Range of Support

We support a wide range of games, from Parkitect, Minecraft, SimAirport, Software Inc, and much much more.
